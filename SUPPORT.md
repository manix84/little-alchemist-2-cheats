# Support Guide 💬

Use this guide to choose the right place for help.

## Questions

For general questions about the app, open a GitHub issue and choose the closest template.

Good questions include:

- "How should this recipe be represented in the data file?"
- "Is this combination missing?"
- "Can the search support this workflow?"

## Bugs 🐛

Open a bug report when the app behaves incorrectly.

Please include:

- Browser and device.
- The element you searched for.
- What you expected.
- What actually happened.
- Screenshots or console errors if useful.

## Recipe Corrections 📚

Open a data correction issue when a Little Alchemist 2 recipe is missing, duplicated, or wrong.

Include the affected element names and the exact combination that should be added, removed, or changed.

## Security 🔒

Do not open public issues for suspected vulnerabilities. Follow [SECURITY.md](SECURITY.md) instead.

## Maintainer Availability

This is a personal open source project, so responses may not be immediate. Clear, focused reports are the easiest to handle.
