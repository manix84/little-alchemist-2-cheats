# Code of Conduct 🤝

## Our Pledge

We want this project to be useful, friendly, and low-friction for everyone who contributes, reports issues, or asks questions.

Everyone participating in this project is expected to:

- Be respectful and constructive.
- Assume good intent while still being clear about problems.
- Keep discussions focused on improving the project.
- Welcome beginners and experienced contributors alike.
- Avoid harassment, insults, personal attacks, and exclusionary language.

## Unacceptable Behavior 🚫

Examples of behavior that is not welcome include:

- Trolling, intimidation, or sustained disruption.
- Public or private harassment.
- Offensive comments related to identity, background, ability, or personal characteristics.
- Publishing someone else's private information without permission.
- Dismissing reports or questions in a hostile or mocking way.

## Enforcement

Project maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this Code of Conduct. Maintainers may also temporarily or permanently restrict participation when needed.

If you see a problem, please report it privately using the contact options in [SUPPORT.md](SUPPORT.md). Include links, screenshots, or context where possible so maintainers can review the situation fairly.

## Scope 🌍

This Code of Conduct applies in project spaces including issues, pull requests, discussions, and any other community venue connected to this repository.
